Project Overview

A 4-DOF servo-driven robotic arm controlled by Arduino Uno. The system allows manual joint control, records movement sequences, and replays them smoothly using interpolation algorithms.

Key Features

Manual Control

Potentiometer-based real-time joint manipulation.

Position Memory

Stores multiple movement steps in memory.

Smooth Playback

Interpolated motion for fluid robotic movement.

Speed Adjustment

Variable playback speed control.

Hardware Components

  • Arduino Uno
  • 4x MG996R Servo Motors
  • 4x Potentiometers
  • 3x Control Buttons (Record / Play / Reset)
  • External 6V Power Supply
  • Robotic Arm Frame

System Working Principle

Technical Highlights

PWM Control

Precise servo actuation using Arduino PWM signals.

Interpolation Algorithm

Gradual angle transitions for smooth motion.

Safety Constraints

Servo angle limits to prevent mechanical damage.
