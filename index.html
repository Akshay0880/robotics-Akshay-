Autonomous Indoor Delivery Robot

Smart logistics automation for campus and small businesses

Problem

Manual internal deliveries are inefficient, time-consuming, and labor-intensive. Small organizations lack affordable automation solutions.

Solution

A low-cost autonomous mobile robot capable of indoor navigation, secure delivery, and real-time tracking via web dashboard.

System Architecture

  • ESP32 for motor control
  • Raspberry Pi for vision processing
  • Firebase backend for real-time monitoring
  • Web dashboard for task management

Features

  • Autonomous navigation
  • Obstacle avoidance
  • Secure PIN-based delivery
  • Live location tracking
  • Task assignment dashboard

Technology Stack

Arduino / ESP32
Raspberry Pi
OpenCV
Firebase
HTML / CSS / JS
